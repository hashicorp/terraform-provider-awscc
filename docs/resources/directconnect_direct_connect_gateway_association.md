Resource Type definition for AWS::DirectConnect::DirectConnectGatewayAssociation

## Schema

### Required

- `associated_gateway_id` (String) The ID or ARN of the virtual private gateway or transit gateway.
- `direct_connect_gateway_id` (String) The ID or ARN of the Direct Connect gateway.

### Optional

- `accept_direct_connect_gateway_association_proposal_role_arn` (String) The Amazon Resource Name (ARN) of the role to accept the Direct Connect Gateway association proposal. Needs directconnect:AcceptDirectConnectGatewayAssociationProposal permissions.
- `allowed_prefixes_to_direct_connect_gateway` (Set of String) The Amazon VPC prefixes to advertise to the Direct Connect gateway. This parameter is required when you create an association to a transit gateway.

### Read-Only

- `association_id` (String) The ID of the Direct Connect gateway association.
- `id` (String) Uniquely identifies the resource.
