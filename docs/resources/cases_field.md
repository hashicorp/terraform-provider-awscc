# awscc_cases_field (Resource)

A field in the Cases domain. This field is used to define the case object model (that is, defines what data can be captured on cases) in a Cases domain.

## Example Usage

```terraform
resource "awscc_cases_field" "example" {
  domain_id = awscc_cases_domain.example.domain_id
  name      = "example-field"
  type      = "Text"

  tags = [{
    key   = "Environment"
    value = "test"
  }]
}

resource "awscc_cases_domain" "example" {
  name = "example-cases-domain-for-field"

  tags = [{
    key   = "Environment"
    value = "test"
  }]
}
```
