# awscc_elementalinference_feed (Resource)

Represents a feed that receives media for inference processing

## Example Usage

### Create an AWS Elemental Inference Feed

Create an AWS Elemental Inference feed for media processing

~> This example is generated by Amazon Bedrock and verified to work using Terraform.

```terraform
resource "awscc_elementalinference_feed" "example" {
  name = "example-inference-feed"

  outputs = [
    {
      name   = "example-output"
      status = "ENABLED"
      output_config = {
        client_configuration = {
          hls_settings = {
            rendition_groups = [
              {
                name                = "example-group"
                rendition_selection = "ALL"
              }
            ]
          }
        }
      }
    }
  ]

  sources = [
    {
      name = "example-source"
      url  = "https://example.com/feed"
    }
  ]

  tags = {
    Environment = "test"
    Purpose     = "example"
  }
}
```
