# awscc_wisdom_knowledge_base (Resource)

Definition of AWS::Wisdom::KnowledgeBase Resource Type

## Example Usage

### Custom Wisdom Knowledge Base with Semantic Chunking

Creates an AWS Wisdom Custom Knowledge Base with semantic chunking configuration, enabling efficient vector-based ingestion with optimized content breakpoints and token management.

~> This example is generated by LLM using Amazon Bedrock and validated using terraform validate, apply and destroy. While we strive for accuracy and quality, please note that the information provided may not be entirely error-free or up-to-date. We recommend independently verifying the content.

```terraform
# Get current AWS region
data "aws_region" "current" {}

# Get current AWS account ID
data "aws_caller_identity" "current" {}

# Example of AWS Wisdom Knowledge Base
resource "awscc_wisdom_knowledge_base" "example" {
  name                = "example-knowledge-base"
  knowledge_base_type = "CUSTOM"
  description         = "Example Wisdom knowledge base created with AWSCC provider"

  # Vector ingestion configuration with semantic chunking
  vector_ingestion_configuration = {
    chunking_configuration = {
      chunking_strategy = "SEMANTIC"
      semantic_chunking_configuration = {
        breakpoint_percentile_threshold = 85
        max_tokens                      = 512
        buffer_size                     = 0.8
      }
    }
  }

  tags = [{
    key   = "ModifiedBy"
    value = "AWSCC"
  }]
}
```
