# awscc_iot_dimension (Resource)

A dimension can be used to limit the scope of a metric used in a security profile for AWS IoT Device Defender.

## Example Usage

```terraform
resource "awscc_iot_dimension" "example" {
  string_values = ["device/+/auth"]
  type          = "TOPIC_FILTER"
  name          = "example"

  tags = [{
    key   = "Modified By"
    value = "AWSCC"
  }]
}
```
