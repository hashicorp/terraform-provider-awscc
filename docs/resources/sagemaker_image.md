# awscc_sagemaker_image (Resource)

Resource Type definition for AWS::SageMaker::Image

## Example Usage

To create a SageMaker Image resource.
```terraform
resource "awscc_sagemaker_image" "example" {
  image_name     = "example"
  image_role_arn = awscc_iam_role.main.arn
  tags = [
    {
      key   = "Name"
      value = "example"
    },
    {
      key   = "Environment"
      value = "Development"
    },
    {
      key   = "Modified By"
      value = "AWSCC"
    }
  ]
}
```
