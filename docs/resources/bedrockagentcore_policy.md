# awscc_bedrockagentcore_policy (Resource)

Resource Type definition for AWS::BedrockAgentCore::Policy

## Example Usage

### Create a BedrockAgentCore Policy
Create a BedrockAgentCore policy with a policy engine and validation settings

~> This example is generated by Amazon Bedrock and verified to work using Terraform.

```terraform
resource "awscc_bedrockagentcore_policy_engine" "example" {
  name        = "ExamplePolicyEngine"
  description = "Example policy engine for BedrockAgentCore"

  policy_statements = {
    statements = [
      {
        statement = <<-EOT
          permit(
            principal,
            action,
            resource is AgentCore::Gateway
          );
        EOT
      }
    ]
  }

  validation_mode = "IGNORE_ALL_FINDINGS"
}

resource "awscc_bedrockagentcore_policy" "example" {
  name              = "ExamplePolicy"
  description       = "Example BedrockAgentCore policy"
  policy_engine_arn = awscc_bedrockagentcore_policy_engine.example.policy_engine_arn
  policy_statements = {
    statements = [
      {
        statement = <<-EOT
          permit(
            principal,
            action,
            resource is AgentCore::Gateway
          );
        EOT
      }
    ]
  }

  validation_mode = "IGNORE_ALL_FINDINGS"
}
```
