# awscc_observabilityadmin_telemetry_enrichment (Resource)

AWS::ObservabilityAdmin::TelemetryEnrichment cloudformation resource enables the resource tags for telemetry feature in CloudWatch to enrich infrastructure metrics with AWS resource tags. For more details: https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/resource-tags-for-telemetry.html

## Example Usage

### Create ObservabilityAdmin Telemetry Enrichment

Configure telemetry enrichment for ObservabilityAdmin to enhance observability data collection and processing

~> This example is generated by Amazon Bedrock and verified to work using Terraform.

```terraform
terraform {
  required_providers {
    awscc = {
      source  = "hashicorp/awscc"
      version = "1.76.0"
    }
  }
}

resource "awscc_observabilityadmin_telemetry_enrichment" "example" {
  # Example configuration for ObservabilityAdmin telemetry enrichment
  tags = {
    Environment = "test"
    Purpose     = "example"
  }
}
```
